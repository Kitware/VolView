# Keyboard shortcuts

## Data management

| Shortcut | Action |
| -------- | ------ |

## Slice

| Shortcut    | Action         |
| ----------- | -------------- |
| Up arrow    | Next Slice     |
| Right arrow | Next Slice     |
| Down array  | Previous Slice |
| Left array  | Previous Slice |

## Mouse Controls

| Action       | 2D              | 3D     |
| ------------ | --------------- | ------ |
| Left         | grayscale       | rotate |
| Mid          | pan             | pan    |
| Right        | zoom            | zoom   |
| Ctrl + Left  | zoom            | zoom   |
| Shift + Left | pan             | pan    |
| Wheel        | navigate slices | zoom   |
